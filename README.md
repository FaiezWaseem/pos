# POS
A Simple Yet Efficient Point of Sale System With Multi-Restaurant Support


## Setup

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# or Build

npm run build

```

## Env

Copy .env.example to .env

```bash
cp .env.example .env
```


## Database

```bash
# Run migrations
php artisan migrate:fresh
```

```bash
# Run seeders
php artisan db:seed
```
